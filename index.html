🧱 MCStructure to Bedrock Commands

แปลง .mcstructure เป็นคำสั่ง setblock/fill สำหรับ NPC ได้ทันที!

📦
ลากไฟล์ .mcstructure หรือคลิกเพื่อเลือกไฟล์
รองรับ Structure Block ทั้งหมด - รองรับ NBT encoding
- ขนาด X
- ขนาด Y
- ขนาด Z
- บล็อกทั้งหมด
- ชนิดบล็อก
- ประหยัดคำสั่ง
💡 วิธีใช้กับ NPC:
1. วางคำสั่งทีละบรรทัดใน NPC Command Block
2. ใช้ /execute @s ~ ~ ~ run [คำสั่ง] นำหน้าแต่ละคำสั่ง
3. โครงสร้างจะ spawn ที่ตำแหน่ง NPC อัตโนมัติ!
